Web developer atau yang terkhusus Fronend developer umumnya banyak menghabiskan waktu dibagian pemodelan atau styling (CSS). Dengan 
pemanfaatan library yang tersedia, kita sekarang memiliki banyak cara untuk lebih cepat dan efektif 
dalam membangun web profile ataupun web aplikasi yang responsif.

Bagi Anda yang mencari librari CSS, di artikel ini kami merangkum library CSS yang bisa Anda gunakan dalam membangun projek Anda.

### Tailwindcss [ðŸš€](https://tailwindcss.com/)

![Tailwindcss](https://res.cloudinary.com/practicaldev/image/fetch/s--QRUnljmI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://dev-to-uploads.s3.amazonaws.com/uploads/articles/mwfwpqtlohyncgc8jvwz.jpg)

TailwindCSS merupakan salah satu library favorit saya yang banyak saya gunakan ketika bekerja dengan projek web.
Library pada umumnya banyak kita dapat terdiri dari komponen pra-desain seperti: button, accordion, select dan sebagainya. Namun
berbeda dengan TailwindCSS dimana kita tidak akan menemukan komponen-komponen seperti itu karena TailwindCSS lebih kesebuah *Utilify first* 
seperti `flex` untuk membuat layout, `p-2` untuk padding, `bg-gray-400` untuk background dengan warna gray dan sebagainya. 
Kalian dapat mempelajarinya di [web resminya](https://tailwindcss.com/).
