[Astro](https://astro.build/) adalah *Static Site Builder* atau penghasil website statik yang lebih moderen dan menjanjikan 
kinerja yang lebih cepat.

Secara default Astro mengirim 0byte javascript, seluruh halaman akan dirender menjadi statik dan menghapus semua javascript pada saat dibuild.
Bearti web kita tidak interkatif dong!. Nah salah satu fitur astro adalah kita dapat menentukan bagian mana yang butuh javascript sehingga kita 
mengirim javascript sesuai kebutuhan.

Walaupun astro masih dalam pengembangan, sudah banyak hal atau fitur-fitur yang ditawarkan sehinggan menjadikan astro salah satu *tool* yang menarik untuk dicoba.

Beberapa fitur yang ada dalam versi beta ini adalah sebagai berikut:

- **Pilih framework favorit kamu**: Kita dapat mengembangankan website menggunakan React, Vue, Svelte, Preact atau sampai dengan sekadar HTML.
- **100% Statik, tanpa JS**: Merender seluruh halaman website anda dalam bentuk HMTL Statik dan menghapus semua file JS.
- **On-Deman JS**: Astro dapat mengirimkan file JS untuk komponen yang membutuhkan interaktif.
- **Fitur cukup lengkap**: Mendukung Typescript, Scope CSS, CSS Module, Sass, Tailwindccss, Md dan Mdx.
- **SEO**: Sitemaps Otomatis.

Ini adalah sekilas pengenalan astro dan fitur-fitur yang dimiliki. Astro cukup populer dikalangan pengembang Front End, melihat dari jumlah star 
di [repositori github](https://github.com/snowpackjs/astro) sudah mencapai 2,9K sejak diperkenalkan dua pekan yang lalu.

Mungkin di artikel berikutnya saya akan membagikan bagaimana cara membuat simple projek seperti blog sederhana menggunakan astro.
